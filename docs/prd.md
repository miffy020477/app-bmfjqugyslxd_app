# 需求文档

## 1. 应用概述

**应用名称**：派出所直通投诉反馈小程序

**应用描述**：为市民提供直接向派出所所长/负责人提交投诉反馈的渠道，支持双向沟通与进度跟踪，简化传统逐级转报流程。市民通过扫描派出所专属二维码进入小程序，无需注册即可提交反馈；所长通过管理后台查看、回复和处理市民反馈。

## 2. 用户与使用场景

**目标用户**：
- 市民：需要向派出所反馈问题或投诉的普通市民
- 所长/负责人：派出所管理人员，负责处理市民反馈

**核心使用场景**：
- 市民在派出所或通过宣传渠道扫描二维码，直接提交投诉或反馈
- 市民查看历史反馈记录，追踪处理进度，与所长进行双向沟通
- 所长登录管理后台，查看待处理反馈，回复市民并更新处理状态

## 3. 页面结构与功能说明

### 3.1 页面结构

```
派出所直通投诉反馈小程序
├── 市民端
│   ├── 反馈提交页
│   ├── 我的反馈列表页
│   └── 反馈详情页
└── 所长管理后台端
    ├── 登录页
    ├── 反馈列表页
    └── 反馈详情页
```

### 3.2 市民端功能说明

#### 3.2.1 反馈提交页
- 市民扫描派出所专属二维码后进入该页面
- 填写反馈内容：
  - 事件描述（文本输入框）
  - 类型选择（下拉或单选，如：治安问题、交通问题、服务投诉、其他）
  - 上传图片（支持多张图片上传）
  - 联系方式（选填，如手机号或姓名）
- 提交反馈后跳转至「我的反馈列表页」

#### 3.2.2 我的反馈列表页
- 展示当前市民提交的所有历史反馈记录
- 每条记录显示：反馈类型、事件描述摘要、提交时间、处理状态（待处理/处理中/已完成）
- 点击某条记录进入「反馈详情页」

#### 3.2.3 反馈详情页
- 展示反馈完整信息：事件描述、类型、上传图片、提交时间、当前状态
- 展示双向沟通记录：
  - 市民的追加留言
  - 所长的回复内容
  - 按时间顺序排列
- 市民可以追加留言（文本输入框+提交按钮）
- 展示处理进度/状态更新记录

### 3.3 所长管理后台端功能说明

#### 3.3.1 登录页
- 所长使用账号密码登录管理后台
- 登录成功后进入「反馈列表页」

#### 3.3.2 反馈列表页
- 展示所有市民提交的反馈记录
- 支持按状态筛选：待处理、处理中、已完成
- 每条记录显示：反馈类型、事件描述摘要、提交时间、市民联系方式（如有）、当前状态
- 点击某条记录进入「反馈详情页」

#### 3.3.3 反馈详情页
- 展示反馈完整信息：事件描述、类型、上传图片、提交时间、市民联系方式（如有）
- 展示双向沟通记录：
  - 市民的留言
  - 所长的回复内容
  - 按时间顺序排列
- 所长可以回复市民（文本输入框+提交按钮）
- 所长可以更新处理状态（下拉选择：待处理/处理中/已完成）
- 状态更新后自动记录更新时间

## 4. 业务规则与逻辑

### 4.1 二维码与派出所关联
- 每个派出所对应一个专属二维码
- 市民扫描二维码后，反馈自动关联至对应派出所
- 所长登录后仅能查看本派出所的反馈记录

### 4.2 市民身份识别
- 市民无需注册，扫码即可使用
- 通过设备标识或临时会话标识关联市民的历史反馈记录
- 市民可选填联系方式，便于所长联系

### 4.3 双向沟通机制
- 市民提交反馈后，可在详情页追加留言
- 所长回复后，市民可在详情页查看回复并继续追加留言
- 每次留言或回复均记录时间戳

### 4.4 状态流转规则
- 初始状态：待处理
- 所长可手动更新为：处理中、已完成
- 状态更新后，市民端同步显示最新状态

### 4.5 数据存储
- 反馈内容、图片、沟通记录、状态更新记录需后端存储
- 市民端和所长端实时同步数据

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 市民未填写事件描述 | 提示「请填写事件描述」，禁止提交 |
| 市民未选择反馈类型 | 提示「请选择反馈类型」，禁止提交 |
| 所长未登录访问管理后台 | 跳转至登录页 |
| 所长登录失败 | 提示「账号或密码错误」 |
| 市民扫描无效二维码 | 提示「二维码无效，请联系派出所」 |
| 市民查看反馈列表为空 | 显示「暂无反馈记录」 |
| 所长查看反馈列表为空 | 显示「暂无反馈记录」 |
| 图片上传失败 | 提示「图片上传失败，请重试」 |

## 6. 验收标准

1. 市民扫描派出所二维码，进入反馈提交页
2. 市民填写事件描述、选择反馈类型、上传图片，提交反馈
3. 市民在「我的反馈列表页」查看刚提交的反馈记录
4. 所长登录管理后台，在「反馈列表页」查看市民提交的反馈
5. 所长点击反馈记录进入详情页，回复市民并更新状态为「处理中」
6. 市民在反馈详情页查看所长回复，追加留言
7. 所长在详情页查看市民追加留言，再次回复并更新状态为「已完成」
8. 市民在反馈详情页查看最新回复和状态更新

- 多派出所统一管理平台
- 反馈评价或满意度调查
- 反馈转派或协同处理功能
- 敏感词过滤或内容审核功能
